# Domain Knowledge

## Primary Domain: Rural Fiber Broadband (Telco/ISP)

GVTC Communications is a cooperative fiber broadband ISP serving the Texas Hill Country (north of San Antonio). Key context:

- **Cooperative model** -- member-owned, not publicly traded. Different incentives than investor-owned telcos.
- **Fiber-to-the-premises** -- not cable, not DSL. Fiber is the product differentiator.
- **Rural service area** -- growth tied to housing development and real estate trends, not just population density.
- **Competitive landscape** -- AT&T Fiber, Spectrum, T-Mobile Home Internet, Starlink all competing for the same homes.

## Telco Terminology I Use Daily

| Term | Meaning |
|------|---------|
| Acquisition | New customers signing up for the first time |
| Retention | Keeping existing customers from leaving |
| Churn | Customers disconnecting/leaving |
| Penetration | % of passings (serviceable homes) that are active customers |
| Passings | Homes that GVTC's fiber network can serve |
| RGU | Revenue-generating unit (each service a customer subscribes to) |
| ARPU | Average revenue per user |
| NPS | Net Promoter Score (customer satisfaction/loyalty metric) |
| SLA | Service level agreement |
| Tier | Speed tier (500 Mbps, 1G, 2G, 5G) |
